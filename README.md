# Financial RAG System

## Architecture Overview

This project implements a financial document intelligence system designed to parse, classify, and structure financial documents for Retrieval-Augmented Generation (RAG) applications.

### Core Components

- **Parser**: Handles document ingestion, preprocessing, heading detection, and hierarchical tree building.
- **Classifier**: Analyzes document content and structure to classify sections or documents using signal-based scoring.
- **Utils**: Shared utilities and constants.

### Usage

Entry point is via `main.py`.
